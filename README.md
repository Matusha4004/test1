# Employee Management Application

## Описание
Приложение на WinForms для отображения и фильтрации списка сотрудников, а также визуализации статистики принятых и уволенных сотрудников за указанный период.

## Особенности

- Отображение списка сотрудников с фильтрами по статусу, отделу, должности и фамилии
- Возможность сортировки таблицы по любому полю
- Построение графика статистики приема и увольнения сотрудников за произвольный период
- Поддержка работы с базой данных SQL Server через Entity Framework и хранимые процедуры
- Использование OxyPlot для визуализации статистики

## Подключение к базе данных
Настройки подключения хранятся в файле `App.config`. Пример секции:

```xml
<connectionStrings>
  <add name="MyConnection" 
       connectionString="Server=.;Database=test1;Trusted_Connection=True;"
       providerName="System.Data.SqlClient" />
</connectionStrings>
```

## Использование приложения

1. Запустите приложение и настройте фильтры
2. Нажмите **Применить**, чтобы обновить таблицу
3. Перейдите в раздел статистики и выберите статус, период и тип (принятые/уволенные)
4. Нажмите **Показать статистику**
5. Наведите курсор и кликните левой кнопкой мыши на любую точку графика, чтобы увидеть точную дату и значение

## Требования

- .NET Framework 4.7.2 или выше
- SQL Server (локально или удалённо)
- OxyPlot.WinForms (через NuGet)

## Запуск и сборка

1. Откройте решение `test1.sln` в Visual Studio
2. Убедитесь, что строка подключения корректна в `App.config`
3. Соберите проект и запустите
